What is LuaPOP3 ?

LuaPOP3 is POP3 client that that can be used to download emails from the mail server. This Library implements most of RFC 1939 commands using the luasocket library.

LuaPOP3 creates the idea of a 'message'. A message is a table with some fields that organize the headers and the body parts, such as text/plain or multipart messages.
The library defines some functions that make it easy to create a mail client application such as webmail or a simple command line tool.

The code is completely written in Lua 5.0 and LuaSocket 2.0 and have 2 simple modules.
The project has started like academic project of 'Programming in Lua' class at PUC-Rio , Brazil

Installation

Just copy the pop3.lua and message.lua to your prefered library path. Be sure that your Lua global envirioment has a table named "socket" from LuaSocket Library.

NOTE : The LuaPOP3 require LuaSocket 2.0 and was tested with the alpha version.

Tests

With the distribution you can find a tests.lua file that show how you can use in a simple way.
You need to change de host, username , login to try with your mail account.

    pop3Client = pop3{
                    host="pop.yourserver.net",
		    user="luapop3@server.net",
		    pass = "luapop"
                    }
It's possible to give the password in a md5 hash string. For this just add the argument secure=true and change the 'pass' argument to the hash string.
About the Luasocket dependence, you need to use the require function to create a "socket" namespace, but just if you don't use the Lua interpreter that came with the distribution .
